" " - Interpreta caracteres especiais

' ' - Não interpreta caracteres especiais

~ Sintaxe ~

read <var1> <var2> <var3>
read <opções> <variável>
Opções
    -p - Para não utilizar o "echo", esse parametro mostra uma frase antes de abrir o campo para o usuário digital, exemplo: read -p "Entre com uma frase: " <var>
    -s - Esse parâmetro serve para não ecoar o que foi digitado (geralmente usado em senhas), exemplo: read -s <var>
    -n - Limita o número de caracteres de entrada "e encerra", exemplo: read -n 4 -p "Digite o pin: " <var>

echo <opções> <argumentos>
Opções
    -e - Nceita caracteres de escape
    -n - Não add nova linha aposta mostrar os argumentos
    -E - Explicitamente nega caractere especial "/"
Caracteres de escape
    \a - Caracter de alerta sonoro (beep)
    \b - Backspace
    \c - Suprime newline, forçando a continuação na mesma linha
    \n - Inicia um nova linha
    \t - Equivale a um espaço de tabulação horizontal
    \v - Equivale a um espaço de tabulação vertical

~ Operador de redirecionamento ~

comando/arquivo > arquivo = Sobreescreve o conteudo
comando/arquivo >> arquivo = Adiciona na última linha do arquivo

~ Varivaveis ~

cria variável - $ var="ola"
cria variável global - $ export var="ola"

~ arrays ~

${arr[*]} - mostra todos os elementos de um array
${arr[indice]} - mostra o indice especifico

arr[0]="ola" - atribui valor a um indice do array
arr=("ola", "mundo", "dois") - atribuir valores simultaneamente

~ PS1 ~

\s - O nome do shell
\u - Nome do usuário que está usando o shell
\h - O hostname
\w - Contém o nome do diretório corrente desde a raiz
\d - Mostra a data no formato: dia_da_semana mês dia
\nnn - Mostra o caracter correspondente em números na base octal
\t - Mostra a hora atual no formato de 24 horas, HH:MM:SS
\T - Mostra a hora atual no formato de 12 horas, HH:MM:SS
\W - Comente nsomente
